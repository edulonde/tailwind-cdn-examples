Hello, Tailwind CSS!

O Tailwindcss é um framework para CSS que oferece algumas utilidades em formato de classe, assim, para estilizar um documento HTML são preciso apenas algumas classes. Todos os exemplos abaixo foram feitos com o Tailwind via CDN. Para usar o Tailwind com cdn não é preciso instalar nada. Mas a melhor maneira para usar o Tailwind é instalando via NPM.
